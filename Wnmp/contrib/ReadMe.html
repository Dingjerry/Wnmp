Wnmp

Welcome to Wnmp!

Wnmp is a distribution of Nginx, MariaDB and PHP distributed with a simple yet feature rich control panel to manage them.

By default, Wnmp installs to C:\Wnmp.

The default login for MariaDB/MySQL is:
Username: root
Password: password

Configuration notes

Default editor

The default editor for Wnmp configuration and log files is 'notepad'. You can set the Editor setting by going to File->Wnmp Options and clicking .. and selecting the location to the editors exe.

Start Wnmp with Windows

The default option for starting Wnmp with Windows is disabled. You can set the Start Wnmp with Windows setting by going to File->Wnmp Options and checking the check box right next to the Start Wnmp on start up label.

Start Nginx on launch

The default option for starting nginx when Wnmp starts is disabled. You can set the Start Nginx on launch setting by going to File->Wnmp Options and checking the check box right next to the Start Nginx on launch label.

Start MariaDB on launch

The default option for starting MariaDB when Wnmp starts is disabled. You can set the Start MariaDB on launch setting by going to File->Wnmp Options and checking the check box right next to the Start MariaDB on launch label.

Start PHP on launch

The default option for starting PHP when Wnmp starts is disabled. You can set the Start PHP on launch setting by going to File->Wnmp Options and checking the check box right next to the Start PHP on launch label.

Minimize Wnmp to tray instead of minimizing

The default option for minimizing Wnmp to the tray when clicking the minimize box is disabled. You can set the Minimize Wnmp to tray instead of minimizing setting by going to File->Wnmp Options and checking the check box right next to the Minimize Wnmp to tray instead of minimizing label.

Minimize Wnmp to tray instead of closing

The default option for minimizing Wnmp to the tray when clicking the X box is disabled. You can set the Minimize Wnmp to tray instead of closing setting by going to File->Wnmp Options and checking the check box right next to the Minimize Wnmp to tray instead of closing label.

Start Wnmp minimized

The default option for starting Wnmp in the minimized state is disabled. You can set the Start Wnmp minimized setting by going to File->Wnmp Options and checking the check box right next to the Start Wnmp minimized label.

Automatically check for updates

The default option for automatically checking for updates is enabled and to check for updates every week. You can set the Automatically check for updates setting by going to File->Wnmp Options and checking the check box right next to the Automatically check for updates label and selecting the desired frequency to check for updates from the combo box.

Nginx Version

The default nginx version is the default included with Wnmp, you have to add desired versions of nginx to C:\Wnmp\nginx-bins\, then going to File->Wnmp Options Nginx and changing the value of the combo box right next to the Nginx Version label and selecting the desired Nginx version.

MariaDB Version

The default MariaDB version is the default included with Wnmp, you have to add desired versions of MariaDB to C:\Wnmp\mariadb-bins\, then going to File->Wnmp Options MariaDB and changing the value of the combo box right next to the MariaDB Version label and selecting the desired MariaDB version.

PHP Processes

The default number of PHP processes is 2 You can set the PHP Processes setting by going to File->Wnmp Options PHP and changing the value of the number box right next to the PHP Processes label and inputing the desired number of PHP processes.

PHP Port

The default PHP port is 9001 You can set the PHP Port setting by going to File->Wnmp Options PHP and changing the value of the number box right next to the PHP Port label and inputing the desired number of PHP port.

PHP Version

The default PHP version is the default included with Wnmp, you have to add desired versions of PHP to C:\Wnmp\php-bins\, then going to File->Wnmp Options PHP and changing the value of the combo box right next to the PHP Version label and selecting the desired PHP version.

Reporting problems

Before you report any problems, please consult the Wnmp FAQ Page and see if your question is already in our list of Frequently Answered Questions (FAQ).

If you cannot find an answer to your question, please feel free to join the google group, at https://groups.google.com/forum/#!forum/wnmp-users. Remember, the more useful information you include in your report, the easier it will be for us to help you!
